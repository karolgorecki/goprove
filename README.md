# goprove

Inspect your project for the best practices listed in the [Go CheckList](https://github.com/matttproud/gochecklist).

### Get Started

    $ go get github.com/karolgorecki/goprove
    $ goprove .

### Usage

```
Usage:

  goprove <directory>

  # output as json
  goprove -output=json <directory>
```
